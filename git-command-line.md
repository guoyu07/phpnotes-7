## Git常用命令


#### 提交|更新一个文件到远程版本库
* 新增一个文件
> git add filename

* 新增所有文件
> git add *

* 将新增的文件提交到HEAD
> git commit -m 'your comment'

* 将本地仓库HEAD中的文件，推送到远程仓库
> git push origin master


#### 查看状态
> git status
